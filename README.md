# Collectividual

Collectividual is a calm editorial-style static website for a psycho-social and cultural project focused on reflection, shared language, symbolic tools, and experimental formats.

## Project structure

- `index.html` — main website
- `style.css` — shared stylesheet for the main website
- `freuds_room.html` — entry page for Freud’s Room
- `freuds_room_bilingual_standalone.html` — standalone Freud’s Room experience

## Current setup

The site is intentionally lightweight:

- static HTML
- separate CSS file
- lightweight inline JavaScript
- mobile-safe layout
- no framework dependency

## Notes

`freuds_room.html` currently redirects to `freuds_room_bilingual_standalone.html` so the main site can stay clean while preserving access to the separate Freud’s Room experience.

## Deployment

This repository is suitable for simple static deployment, including GitHub Pages.
